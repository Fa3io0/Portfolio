Console Dungeons: RPG de Combate por Turnos en C#

Este proyecto esta desarrollado en C# como una aplicación de consola, subido a un repositorio de GitHub, es un juego de rol (RPG) de combate por turnos que simula enfrentamientos entre un jugador y varios tipos de enemigos controlados por la CPU. El código implementa principios de la Programación Orientada a Objetos (POO), utilizando clases base para el personaje del jugador y los enemigos.
